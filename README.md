# 🎧 YouTube to MP3 Converter

![Python](https://img.shields.io/badge/Python-3.12+-blue)
![yt-dlp](https://img.shields.io/badge/yt--dlp-latest-green)
![FFmpeg](https://img.shields.io/badge/FFmpeg-required-orange)
![Status](https://img.shields.io/badge/status-in%20development-yellow)

Aplicación en Python que permite descargar el audio de un video de YouTube y convertirlo a formato MP3.

---

## ✨ Características

* 🎵 Descarga audio desde YouTube
* 🔄 Conversión automática a MP3
* 📁 Guardado en carpeta predeterminada
* ⚡ Uso simple desde consola

## ⚙️ Instalación

### 🔹 1. Clonar el repositorio

```bash
git clone https://github.com/tu_usuario/tu_repo.git
cd tu_repo
```

---

## 🍎 Mac (Homebrew)

```bash
/opt/homebrew/bin/python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
brew install ffmpeg
```

---

## 🪟 Windows

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

### Instalar FFmpeg

1. Descargar: https://ffmpeg.org/download.html
2. Agregar al PATH
3. Verificar:

```bash
ffmpeg -version
```

---

## ▶️ Uso

```bash
python convertidor.py
```

Luego pega el enlace del video de YouTube cuando lo solicite.

---

## 📁 Ubicación de descarga
